Wrincle App - Capstone Project
The wrestling app that we built was designed to collect match information and convert it into a CSV file for website upload. Our team used React Native to develop the app, and Expo Go as the mobile app development tool, Visual Studio Code as the integrated development environment (IDE), and GitHub as the version control system. Our wrestling app simplified match data collection for teams by providing a user-friendly interface for inputting wrestler names, weights, and match events and outcomes. It ensured accurate and structured data for website upload, enabled teams to make more informed decisions, track performance, and improve overall outcomes.
